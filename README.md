# Talent Directory App

A full-stack app to manage talent profiles.

---

## Setup & Run

1. **Clone Repo**  
```bash
git clone https://github.com/dakshg393/talentDirectory.git

2.Backend

cd backend
npm install
# create .env with PORT and MONGO_URI
npm run dev

3.Frontend
cd frontend
npm install
# create .env with VITE_SERVER_URL=http://localhost:5000
npm run dev
